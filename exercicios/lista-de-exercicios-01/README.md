## 📌 Sobre
Este projeto reúne 18 exercícios desenvolvidos em Java, com foco na prática de lógica de programação, cálculos matemáticos e entrada e saída de dados.

Os enunciados de cada exercício foram adicionados diretamente nas respectivas classes `.java`.

## 🗂️ Estrutura
O projeto foi organizado em um único projeto Java chamado `trabalhopratico01`, contendo 18 classes, sendo uma para cada exercício proposto.

Exemplo de organização:
- `TP01Ex01.java`
- `TP01Ex02.java`
- `TP01Ex03.java`
- `...`
- `TP01Ex18.java`

## 📚 Conteúdos abordados
Ao longo desta lista de exercícios, são trabalhados temas como:
- entrada e saída de dados
- variáveis e operadores
- cálculos matemáticos
- fórmulas geométricas
- média aritmética e geométrica
- conversão de unidades
- lei de Ohm
- temperatura
- trigonometria
- potenciação

## 🎯 Objetivo
Praticar a implementação de algoritmos em Java, reforçar conceitos básicos da linguagem e organizar a resolução dos exercícios de forma clara e individual.
