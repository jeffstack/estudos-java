## 📌 Sobre
Neste repositório ficam exercícios, testes e pequenos projetos desenvolvidos durante meu aprendizado da linguagem Java.

## 🗂️ Estrutura
Este repositório está organizados em pastas, separadas entre exercícios e mini-projetos.

## 📚 Conteúdos abordados
Ao longo do repositório, podem aparecer temas como:
- lógica de programação
- métodos
- vetores e matrizes
- programação orientada a objetos
- entrada e saída de dados
- classes e objetos

## 🎯 Objetivo
Registrar minha evolução nos estudos em Java, praticar conceitos importantes da linguagem e manter os arquivos organizados para consultas futuras.
